What the hell I'm looking at?

This is a little project I've done to learn a little bit of python:

    This is a list of the routes and what you can do on them by using POSTMAN

  • A GET to "/" will bring you this HTML Static file. There is not any other possible petition type to "/".

  • A GET to "/games will bring you a JSON with all the Games on the DB (JSON file on the Static folder).

  • A GET to "/games/{GameId}" will bring you a JSON with the solicited game if the GamesId exist

  • A POST to "/games/{GameId}" with a proper JSON object would let you, as expect, sabe a new Game to the DB.

  • A PUT to "/games/{GameId}" with a proper JSON object would let you update a new Game to the DB.

  • A DELETE to "/games/{GameId}" will Delete the game which GameId is related to.
    This can only be undone by downloading again the repo.
